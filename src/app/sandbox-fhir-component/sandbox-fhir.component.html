• FHIR Server

    This is an implementation of the FHIR standard using the US Core Implementation guide. We expect client developers to use the server as part of their development activities to work with different data sets.

  • FHIR Query Tool

    FHIR Query Tool can be used to test a FHIR Server using various search parameter combinations. The tool makes it easy for users to query FHIR servers without knowing the specific authorization workflows and FHIR syntaxes for searches.

  • FHIR Conformance Test Tool

    The FHIR Conformance Test Tool provides an easy way for servers to test their conformance to an implementation guide using test data and test scripts.

  • FHIR Bulk Data API

    The FHIR Bulk Data API is an implementation of the FHIR Bulk Data proposal.
