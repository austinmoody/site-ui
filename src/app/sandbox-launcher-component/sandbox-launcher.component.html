Welcome to SITE

The Standards Implementation & Testing Environment (SITE) is a centralized collection of testing tools and resources designed to assist health IT developers and health IT users fully evaluate specific technical standards and maximize the potential of their health IT implementations. SITE is organized in a collection of sandboxes that provide test tools, sample data, collaboration resources, and useful links.

  • C-CDA Sandbox

    This sandbox contains resources and test tools related to the Consolidated Clinical Document Architecture standard.

  • Direct Transport Sandbox

    The sandbox contains resources and test tools related to the Direct project and Direct systems registered by health IT developers that can be used by implementers to test interoperability.

  • FHIR Sandbox

    This sandbox contains resources and test tools related to the Fast Healthcare Interoperability Resources. (FHIR) standard.

  • Quality Reporting Document Architecture-Clinical Quality Measures Sandbox

    This sandbox contains resources and tools related to evaluating the accuracy of clinical quality measure calculations with Cypress.
